Premiers pas sur votre espace

Cette introduction est destinée aux responsables d'associations. Si vous êtes simplement membre d'une association qui dispose d'un espace RunOrg, nous vous conseillons de lire « Premier pas pour les membres »

Bienvenue ! Vous venez de créer ou d'installer un espace RunOrg pour votre association ou votre organisation (si ce n'est pas le cas, cliquez ici pour le faire). Cette courte introduction va vous aider à partir du bon pied dans l'utilisation de votre espace.
